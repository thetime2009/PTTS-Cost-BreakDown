ขั้นตอนที่ 1 — ติดตั้ง Google Apps Script
───────────────────────────────────────────
1. เปิด Google Sheet ของคุณ
   https://docs.google.com/spreadsheets/d/1zn4U7nsqM5M_fePxzqPGGMW3MX16XEPl6ZHR9-jZOUo/edit

2. เมนู: Extensions → Apps Script

3. ลบโค้ดเดิมทั้งหมดในหน้าต่าง (ถ้ามี)

4. คัดลอกโค้ดจากไฟล์ Code.gs แล้ววางลงไป

5. บันทึก (Ctrl+S) — ตั้งชื่อ Project เป็น "PTTS-Form" หรืออื่นๆ

ขั้นตอนที่ 2 — Deploy เป็น Web App
───────────────────────────────────────────
1. คลิกปุ่ม "Deploy" (มุมขวาบน) → "New deployment"

2. ตั้งค่าดังนี้:
   - Type: Web app
   - Description: PTTS Form v1
   - Execute as: Me (ชื่อบัญชี Google ของคุณ)
   - Who has access: Anyone  ← สำคัญมาก!

3. คลิก "Deploy"

4. อนุญาตสิทธิ์ (Authorize) เมื่อระบบถาม
   → Allow → Allow

5. คัดลอก "Web app URL" ที่ขึ้นมา
   (รูปแบบ: https://script.google.com/macros/s/XXXXX/exec)

ขั้นตอนที่ 3 — ใส่ URL ในเว็บฟอร์ม
───────────────────────────────────────────
1. เปิดไฟล์ form.html ใน Browser

2. ในกล่องสีเหลืองด้านบน วาง URL ที่คัดลอกมา

3. กดปุ่ม "💾 บันทึก URL"

4. พร้อมใช้งาน! กรอกข้อมูลและกด "บันทึกลง Google Sheets"

════════════════════════════════════════════════════════
  หมายเหตุ
════════════════════════════════════════════════════════
• ข้อมูลจะบันทึกลงใน tab "DATA" โดยอัตโนมัติ
• ถ้า tab ชื่อต่างออกไป ให้แก้ SHEET_NAME ใน Code.gs
• URL จะถูกจำไว้ใน Browser (localStorage) ไม่ต้องใส่ซ้ำ
• กรณีเปลี่ยน Deploy ใหม่ ให้อัปเดต URL ในฟอร์มด้วย
• Dropdown วัสดุ (MAT) โหลดจาก tab "MAT" คอลัมน์ A
  — ถ้าต้องการเพิ่ม/ลดวัสดุ ให้แก้ข้อมูลใน tab MAT
  — กด "💾 บันทึก URL" อีกครั้งเพื่อรีโหลด MAT
════════════════════════════════════════════════════════

⚠️  หลังอัปเดต Code.gs ให้ Deploy ใหม่ทุกครั้ง
    (Deploy → Manage Deployments → Edit → Version: New → Deploy)

วิธีเปลี่ยนโลโก้
─────────────────
1. เปิด ดูแบบเสนอราคา → ⚙️ ตั้งค่าบริษัท
2. คลิกกรอบโลโก้ (มุมซ้ายบนของ modal)
3. เลือกไฟล์รูป (PNG, JPG — แนะนำสี่เหลี่ยมจัตุรัส)
4. กด 💾 บันทึก

โลโก้จะอัปเดตพร้อมกันทุกจุด:
  • Header บนสุดของแอป
  • Sidebar เมนูซ้าย
  • ใบเสนอราคา (Official Quotation)

การลบโลโก้
───────────
  กด 🗑 ลบ ในหน้าตั้งค่าบริษัท
  → ทุกจุด fallback กลับเป็นไฟล์ PTS.png อัตโนมัติ

ที่เก็บข้อมูล
─────────────
  โลโก้ถูกบันทึกเป็น base64 ใน localStorage (ptts_logo_b64)
  ไม่ต้องอัปโหลดซ้ำทุกครั้งที่เปิดแอป
  ⚠ ถ้าล้าง browser cache/localStorage โลโก้จะหาย
    → ต้องอัปโหลดใหม่อีกครั้ง

ข้อมูลทั้งหมดแสดงในใบเสนอราคา ได้แก่:
  • ชื่อบริษัท (ภาษาไทย / อังกฤษ)
  • ที่อยู่
  • โทรศัพท์ / อีเมล์
  • เลขที่ผู้เสียภาษี (TAX ID)

วิธีแก้ไข
──────────
  ดูแบบเสนอราคา → ⚙️ ตั้งค่าบริษัท → แก้ไข → บันทึก

ที่เก็บข้อมูล
─────────────
  บันทึกใน localStorage (ptts_company_cfg)
  ⚠ ถ้าล้าง localStorage ข้อมูลจะกลับเป็นค่าเริ่มต้น

  ฟีเจอร์: ค้นหาละเอียด (Advanced Search) ในแท็บ DATA
════════════════════════════════════════════════════════

ตัวกรองนอกแผงค้นหา (แถวบนตาราง)
────────────────────────────────────
  [สถานะ]              — กรองตาม: ผ่าน / อนุมัติ / รอสรุป / ยกเลิก
  [กรองตรง OD×ID×H]   — ติ๊ก checkbox → ใช้ค่า OD, ID, H จากฟอร์มปัจจุบัน
                          label จะแสดงค่าเช่น (OD200×ID42×H300)
                          กรอกแค่บางตัวก็ได้ (เช่น OD อย่างเดียว)
  [กรองตาม OD]         — กรองเฉพาะ OD จากฟอร์ม (ไม่สนใจ ID, H)

แผงค้นหาละเอียด (กด "🔎 ค้นหาละเอียด")
─────────────────────────────────────────
  OD min/max       — ช่วง OD (mm)
  วันที่ from/to   — ช่วงวันที่บันทึก
  ราคาเสนอขาย     — ช่วงราคา (฿/ลูก)
  MAT ฝา          — พิมพ์บางส่วน เช่น "SPCC" หรือ "0.6"
  MAT ตะแกรง      — พิมพ์บางส่วน เช่น "L-5"
  แบบงาน          — พิมพ์บางส่วน เช่น "หัว-ท้าย"

  badge "ตัวกรองที่ใช้" แสดงจำนวน filter ที่ active
  ปุ่ม ✕ ล้างตัวกรอง → ล้างทุก filter พร้อมกัน

จำนวนแถวต่อหน้า: 50 รายการ

════════════════════════════════════════════════════════
  ฟีเจอร์: ODฝา2 (ฝา 2 ขนาด)
════════════════════════════════════════════════════════

เมื่อไหร่ใช้
─────────────
งานบางชิ้นมีฝาบน/ล่างขนาด OD ต่างกัน เช่น
  ODฝา2 / ODฝา1 × ID × H × จำนวน
  เช่น  100/80 × 42 × 200 × 5 ea

วิธีใช้งาน
─────────────
1. ในแท็บ ฟอร์ม → กรอก OD, ID, H, จำนวน ตามปกติ
2. ติ๊ก checkbox "มีฝา 2 ขนาด (ODฝา2)"
   → กล่องสีเหลืองจะเปิดขึ้น
3. กรอก OD ฝา2 (mm)
4. เลือก MAT ฝา2 จาก dropdown (ใช้รายการเดียวกับฝาบน/ล่าง)
5. เลือกโหมดการคำนวณ:
   ● คำนวณ  — ระบบดึงราคา MAT + คิดพื้นที่ฝา2 แล้วรวมเข้าต้นทุนอัตโนมัติ
   ○ ไม่คำนวณ — บันทึก OD ฝา2 ไว้อ้างอิง/แจ้งราคา แต่ไม่รวมในต้นทุน
6. ช่อง "Cost ฝา2 (฿/ลูก)" จะคำนวณอัตโนมัติถ้าเลือก "คำนวณ"
   หรือกรอกเองได้ถ้าต้องการปรับ

การคำนวณต้นทุนฝา2 (โหมด คำนวณ)
──────────────────────────────────
  พื้นที่แผ่น = ความกว้าง × ความยาว (จาก MAT)
  พื้นที่ฝา2  = π × (OD2 + ช่องว่างจีบ)² / 4
  จำนวน/แผ่น  = พื้นที่แผ่น ÷ พื้นที่ฝา2  (ปัดลง)
  Cost/ลูก     = ราคา MAT ÷ จำนวน/แผ่น

  ✔ ใช้ค่า "ช่องว่างจีบ" (f_gapWeld) เดียวกับฝาบน/ล่าง
  ✔ ราคา MAT ดึงจากตาราง MAT ที่ตั้งค่าไว้

การบันทึกลง Google Sheets
──────────────────────────
  คอลัมน์ BC (54) — OD ฝา2 (mm), 0 = ไม่มีฝา2
  คอลัมน์ BD (55) — รหัส MAT ฝา2
  คอลัมน์ BE (56) — Cost ฝา2 / ลูก
  คอลัมน์ BF (57) — Flag: 1 = คำนวณ, 0 = ไม่คำนวณ

  ⚠ ไม่ต้อง Redeploy Code.gs — ต่อท้ายแถวข้อมูลเดิม

การค้นหาในแท็บ DATA
──────────────────────
  พิมพ์ค้นหาด้วย: OD ฝา2 (ตัวเลข) หรือ รหัส MAT ฝา2
  ระบบจะกรองแถวที่มี ODฝา2 ตรงกันออกมา
